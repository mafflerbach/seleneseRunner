Install

The SeleneseRunner use composer vor loading requirements. At the first time call php composer install

Running tests in commandline

Start simple php run.php -c [config.json] your tests. You can overwrite your config file with follow additional parameters:

  • -d [directory] path to test directory
  • -i [imagedirectory] path to image directory vor captured screenshots
  • -s [seleniumhub][http://localhost:4444/wd/hub] reset the target selenium hub.
  • -username username for saucelabs connection
  • -access_key username for saucelabs access_key
  • -saucelabsSelenium username for saucelabs Selenium hub
  • -local if you would like to run your test local, even though you have set the saucelabs credentials
